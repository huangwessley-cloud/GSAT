# 墨跡 — 個人部落格

- `index.html` — 部落格前台
- `admin.html` — 後台 CMS(寫文章、發佈、管理)

## 重要限制

後台資料儲存在**瀏覽器的 localStorage**,不是雲端資料庫。這代表:

- 你在自己電腦的瀏覽器用 `admin.html` 寫的文章,只有在**同一台電腦、同一個瀏覽器**打開 `index.html` 才看得到。
- 部署到網路上後,**別人打開你的網站不會看到你發佈的文章**,因為他們的瀏覽器裡沒有這份 localStorage 資料。
- 如果想讓所有訪客都看到同樣的文章內容,必須：
  1. 在 `admin.html` 的「設定」頁面用「匯出 JSON」把文章資料匯出，
  2. 手動把匯出的文章內容貼回 `index.html` 裡的 `defaultPosts` 陣列，
  3. 重新部署。

這是純前端、無後端資料庫架構下的已知限制，不是 bug。若之後想要「每個人都能即時看到最新文章」，需要接上真正的後端（例如一個小型 API + 資料庫）。

## 本機預覽

直接用瀏覽器打開 `index.html` 或 `admin.html` 即可，不需要伺服器。
