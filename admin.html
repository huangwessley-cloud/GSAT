資料備份與同步

資料

匯出文章備份(JSON)
將所有文章(含草稿)下載為 JSON 檔,可用於備份或搬移到其他瀏覽器。
匯入文章備份
從先前匯出的 JSON 檔還原文章。匯入會覆蓋目前的所有資料。
還原範例文章
清除所有文章,還原為八篇內建範例。此動作無法復原。

與部落格前台同步

後台的資料儲存在瀏覽器本機儲存空間中。只要用同一個瀏覽器開啟 index.html,前台就會自動顯示這裡「已發佈」的文章;草稿不會出現在前台。若要換瀏覽器或電腦,請先匯出 JSON,再到新環境匯入。
